# Example Videos

This directory is for video before/after comparison examples.

## Adding Video Examples

To add video comparisons:
1. Use short clips (5-10 seconds)
2. Provide both before and after versions
3. Use consistent naming: `model-before.mp4`, `model-after.mp4`
